¿Qué es Gamers Den?

Gamers Den se trata de un proyecto de una aplicación web basado en un comparador de precios de distintos juegos. Dichos juegos mediante aplicaciones y algoritmos busca los mejores precios del mercado y los compara por ti. Es una forma sencilla y rápida que permite la búsqueda del mejor precio, permitiendo al usuario ahorrar tiempo, esfuerzo y dinero. En la página web, un usuario puede registrarse para poder interactuar con otros usuarios y administradores creando una comunidad interactiva. Además, dicho usuario podrá tener acceso a contenidos como eventos, noticias, novedades, información de videojuegos, tienda, lista de deseos, etc... Las distintas funciones u operaciones a cumplimentar son las siguientes:

1)Comparador de videojuegos

El usuario tendrá toda la información sobre cada juego (descripción, fecha de lanzamiento,sitios donde se pueden comprar) Por tanto, cuando un usuario busque un videojuego le saldrán las características de ese juego y los precios en las diferentes páginas que se pueda comprar.

2)Foro

3)Tienda de segunda mano

4)Noticias

5)Calendario

6)Interfaz social

7)Ranking

En cuanto a los usuarios, pueden haber 5 usuarios diferentes. Según el tipo de usuario tendrá diferentes accesos o servicos:

1)Usuario Registrado

2)Moderador

3)Administrador

4)Catalogador

5)Visitante
